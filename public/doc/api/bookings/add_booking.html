Bookings API

Add booking

POST /api/bookings

Parameters

Name Description
first_name first name of customer
last_name last name of customer
email email of customer
mobile mobile number of customer
checkin checkin date of booking, can not be less than today
checkout checkout date of booking, can not be less than today and checkin
arrival_time arrival time of booking
care_id care id of care for which booking is created
user_id id of user who is creating booking
relationship_id get relationship ids from relationships api
no_of_guests number of guests
bed_bookings_attributes bed ids for which booking is created
